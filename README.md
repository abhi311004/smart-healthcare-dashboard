# 📊 smart-healthcare-dashboard

## 📖 Project Description
This project is a Power BI dashboard that analyzes patient wait list data in the healthcare system. It provides insights into wait times, case types, specialties, and trends to help improve healthcare service efficiency.

---

## 🎯 Key Features
- KPI metrics for latest and previous year wait list
- Case type analysis (Outpatient, Day Case, Inpatient)
- Age group vs time band analysis
- Top 5 specialties based on wait list
- Monthly trend analysis of patient cases
- Comparison of Day Case, Inpatient, and Outpatient trends
- Interactive filters for:
  - Archive Date
  - Case Type
  - Specialty Name

---

## 🛠️ Technologies Used
- Power BI
- Data Visualization
- DAX

---

## 📊 Key Insights
- Latest month wait list is **630K**, higher than previous year (**586K**)
- Outpatients contribute the highest share of cases (~71%)
- Most patients fall under **0–3 months and 18+ months** wait time
- Certain specialties like Ophthalmology show significant wait lists
- Monthly trends show gradual increase in outpatient cases

---

## 🚀 How to Use
1. Download the Power BI (.pbix) file
2. Open it in Power BI Desktop
3. Use filters (Date, Case Type, Specialty)
4. Analyze charts and KPIs for insights
